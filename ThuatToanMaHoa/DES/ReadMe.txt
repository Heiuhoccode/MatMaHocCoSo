Mã hóa và Giải mã file âm thanh, file hình ảnh, file văn bản theo thuật toán DES
***
File âm thanh chuẩn bị: HappyBirthday.wav
File hình ảnh chuẩn bị: input.png
File văn bản chuẩn bị: input.txt
File mã hóa: Encode_sound.py
File giải mã: Decode_sound.py
***
MÃ HÓA
Để mã hóa file âm thanh cần chạy file mã hóa "Encode_sound.py".
Kết quả cho ra 1 file âm thanh mã hóa: "Encode_sound.wav" và 1 file nhị phân "Encode_binary_sound.txt".
***
GIẢI MÃ
Để giải mã file âm thanh "Encode_sound.wav" cần chạy file mã hóa "Decode_sound.py".
Kết quả cho ra 1 file âm thanh giải mã: "Decode_sound.wav" và 1 file nhị phân "Origin_binary_sound.txt".
File âm thanh "Decode_sound.wav" có nội dung giống với "HappyBirthday.wav".
